투썸
==================
## 00. 소개
    라즈베이파이, 아두이노, 안드로이드를 이용하여 IoT 환경 구축하기
    수면 시 심박수가 10~15% 감소하는 점을 이용하여 수면 상태 시 연결된 전자제품의 전원을 종료한다.

## 01. 언어 및 기술
* JAVA (JDK 17)  
* MariaDB 10  
* Android Studio - SDK 29
* Python3 - pygatt(BLE)
* PHP
* C++ - DFRobot_Heartrate(심박센서)

## 02. 개발 기간 및 인원
* 기간: 2020/05/08 ~ 2020/06/17  
* 인원: 3명
